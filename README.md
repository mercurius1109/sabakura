# Survival Craft

Vue.js + Tailwind CSS + Vite で動く、ブラウザ向けサバイバルクラフト管理UIのプロトタイプです。

## セットアップ

```powershell
pnpm install
pnpm dev
```

`npm` を使う場合:

```powershell
npm install
npm run dev
```

## 主なファイル

- `src/App.vue`: ゲームUIとクラフトロジック
- `src/main.js`: Vueアプリの起動
- `src/style.css`: Tailwind読み込みとベースCSS
- `tailwind.config.js`: Tailwindテーマ設定
